# MUSIC PLAYER
A command-line music player built in Python.
# What it does right now:
-> Downloads audio from YouTube using yt-dlp
-> Plays, pauses, and resumes songs from a folder
-> Navigate songs with simple commands (play, pause, resume, next, quit)
# How to run:
Activate the venv(virtual environment) you can do this with:
python3 -m venv venv, run player.py, type commands in the terminal.
# Setup
1. Clone the repo
2. python3 -m venv venv
3. source venv/bin/activate
4. pip install yt-dlp pygame
